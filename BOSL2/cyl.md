# cyl() / xcyl/ycyl/zcyl / cylinder() — детальный справочник

> Раздел: [BOSL2 — справочник](../BOSL2.md) › 3D-примитивы.
> Источник: `shapes3d.scad`. Модуль и функция (`vnf = cyl(...)`).

Цилиндр (и конус) со скруглением/фаской торцов, текстурой поверхности и поддержкой анкеров.

## Использование

```openscad
// Базовый
cyl(l|h|length|height, r, [r2], [center|anchor=], [circum=], [realign=]);
cyl(l|h=, d=);
cyl(l|h=, r1=|d1=, r2=|d2=);
// С фаской
cyl(..., chamfer=, [chamfang=], [from_end=]);
cyl(..., [chamfer1=], [chamfer2=], [chamfang1=], [chamfang2=], [from_end=]);
// Со скруглением торцов
cyl(..., rounding=, [teardrop=], [clip_angle=]);
cyl(..., [rounding1=], [rounding2=], [teardrop=], [clip_angle=]);
// С текстурой поверхности
cyl(..., texture=, [tex_size=]|[tex_reps=], [tex_depth=], [tex_rot=],
        [tex_samples=], [tex_taper=], [tex_inset=], [style=]);
```

## Параметры

| Параметр | По умолч. | Назначение |
|----------|-----------|-----------|
| `l`/`h`/`length`/`height` | 1 | Длина вдоль оси ориентации. |
| `r` | 1 | Радиус. `r1`/`r2` — низ/верх (для конуса). |
| `d`/`d1`/`d2` | — | Диаметр (вместо r). |
| `circum` | `false` | `true` → цилиндр **описывает** заданную окружность; иначе вписан. Влияет на `realign`. |
| `realign` | авт. | `false` → вертикальное ребро на X+; `true` → вертикальная грань на X+. |
| `chamfer`/`chamfer1`/`chamfer2` | нет | Размер фаски торцов (общий/низ/верх). |
| `chamfang`/`chamfang1`/`chamfang2` | авто | Угол фаски от торца (по умолч. посередине между торцом и боковой поверхностью). |
| `from_end`/`from_end1`/`from_end2` | `false` | Если `true` — фаска отмеряется вдоль боковой грани от торца, а не как катет от ребра. |
| `rounding`/`rounding1`/`rounding2` | нет | Радиус скругления торцов. |
| `teardrop` | `false` | Ограничить скругление **нижнего** ребра углом от горизонтали (число = градусы, `true` = 45°). Для печатаемых отверстий без поддержек. |
| `clip_angle` | 90 | Клиппинг дуги скругления нижнего ребра (90 = полная окружность, 0 = убрать целиком). |
| `extra`/`extra1`/`extra2` | 0 | Невидимая добавка длины с торцов (для корректного `difference`). |
| `texture` | — | Имя текстуры (`"ribs"`, `"diamonds"`, `"dimples"`, …), массив высот или VNF-тайл. См. [BOSL2.md §6.6](../BOSL2.md). |
| `tex_size` | `[5,5]` | Целевой размер тайла. Либо `tex_reps=` — число повторов. |
| `tex_depth` | 1 | Глубина текстуры (отрицательная → инверсия). |
| `tex_rot` | 0 | Поворот текстуры (кратный 90°). |
| `tex_taper` | нет | Заводить глубину текстуры в ноль на концах (доля высоты, lookup или функция). |
| `tex_inset` | `false` | Утопить текстуру в поверхность на долю глубины. |
| `style` | `"min_edge"` | Стиль триангуляции heightfield-текстур. |
| `anchor` | **`CENTER`** | См. [attachments.md](./attachments.md). |
| `spin` | 0 | Поворот вокруг Z. |
| `orient` | `UP` | Ось цилиндра. |

## Ориентированные варианты (без `rot`)

| Модуль | Эквивалент | Особенность |
|--------|-----------|-------------|
| `xcyl(l, r|d=, ...)` | `cyl(..., orient=RIGHT)` | Ось вдоль X. **Не принимает `orient`.** Анкеры в мировых осях. |
| `ycyl(l, r|d=, ...)` | `cyl(..., orient=BACK)` | Ось вдоль Y. |
| `zcyl(l, r|d=, ...)` | `cyl(..., orient=UP)` | Ось вдоль Z (обычный). |

> **Правило:** не делайте `zrot(90) cyl(...)`, чтобы положить цилиндр на бок — используйте `xcyl()`/`ycyl()`.

## cylinder() — нативная обёртка

`cylinder(h, r|d=, [center])` расширяет встроенный `cylinder()`: добавляет анкеры/attach и форму-функцию `vnf = cylinder(...)`.
**Важно:** `cylinder()` по умолчанию `anchor=BOTTOM` (как нативный). Для BOSL2-функций берите `cyl()` (по умолчанию `CENTER`).

## Примеры

```openscad
include <BOSL2/std.scad>

// Цилиндр с скруглёнными торцами; низ печатается без поддержек
cyl(h=30, d=20, rounding=3, teardrop=true);

// Фаска только сверху
cyl(h=30, d=20, chamfer2=2);

// Отверстие-«капля» под болт (вычитаем): teardrop ограничивает свес
diff() {
  cuboid([40,40,10]);
  tag("remove") zcyl(h=50, d=4);   // сквозное отверстие
}

// Текстурированный цилиндр («diamonds»)
cyl(h=40, d=30, texture="diamonds", tex_size=[5,5], tex_depth=0.5);

// Сопло (конус) со скруглением
cyl(h=20, d1=20, d2=8, rounding1=2);
```

## Частые ошибки

* Не путайте `cyl` (CENTER) и `cylinder` (BOTTOM) при позиционировании.
* `teardrop` влияет только на **нижнее** ребро; для верхнего используйте `clip_angle` или ориентацию.
* Текстура несовместима с `rounding`/`chamfer` одновременно — обработайте торцы отдельно.
