# cuboid() / cube() — детальный справочник

> Раздел: [BOSL2 — справочник](../BOSL2.md) › 3D-примитивы.
> Источник: `shapes3d.scad`. Модуль и функция (`vnf = cuboid(...)`).

Прямоугольный параллелепипед со скруглением/фаской выбранных рёбер и граней.
Это **рабочая лошадка** для корпусов, кронштейнов, оснований.

## Использование

```openscad
cuboid(size, [anchor=], [spin=], [orient=]);
cuboid(size, p1=, ...);            // один угол в p1, forces anchor=FRONT+LEFT+BOTTOM
cuboid(p1=, p2=, ...);             // по двум противоположным углам

// С фаской:
cuboid(size, [chamfer=], [edges=], [except=], [trimcorners=], ...);
// Со скруглением:
cuboid(size, [rounding=], [teardrop=], [edges=], [except=], [trimcorners=], ...);
```

## Параметры

| Параметр | По умолч. | Назначение |
|----------|-----------|-----------|
| `size` | 1 | Размер: скаляр (куб) или `[x,y,z]`. |
| `chamfer` | нет | Размер фаски (вдоль граней). |
| `rounding` | нет | Радиус скругления рёбер. |
| `edges` | `"ALL"` | Какие рёбра обрабатывать. См. систему ниже и [attachments.md](./attachments.md). |
| `except` | нет | Рёбра, которые **исключить** из набора. |
| `trimcorners` | `true` | Скруглять/фасочить углы, где сходятся 3 выбранных ребра. |
| `teardrop` | `false` | Ограничить скругление нижнего ребра углом от вертикали (число = градусы, `true` = 45°). Для печати без поддержек. |
| `clip_angle` | нет | Как `teardrop`, но скругление заканчивается в основании (угол от вертикали; 90 = полная окружность). |
| `p1` | — | Угол объекта; принудительно `anchor=FRONT+LEFT+BOTTOM`. |
| `p2` | — | Вместе с `p1` задаёт куб по двум углам. |
| `anchor` | **`CENTER`** | См. [attachments.md](./attachments.md). |
| `spin` | 0 | Поворот вокруг Z. |
| `orient` | `UP` | Куда смотрит верх объекта. |

## Ключевые правила

1. **`chamfer` и `rounding` взаимоисключающи** — только один тип обработки рёбер за вызов.
2. **Отрицательные значения** (`chamfer=-5`, `rounding=-5`) создают внешние галтели (фасонный переход наружу), но применяются **только к рёбрам вокруг верхней/нижней граней** (`TOP`/`BOT`). С набором рёбер, отличным от `"ALL"`, при отрицательном значении будет ошибка.
3. **По умолчанию `anchor=CENTER`** — объект центрирован (диапазон `±size/2`). Это отличается от нативного `cube()`. Чтобы поставить на плоскость: `anchor=BOT` или `anchor=FRONT+LEFT+BOT`.

## Система `edges=` / `except=` (кратко)

Каждый дескриптор ребра — это вектор направления, строка `"X"`/`"Y"`/`"Z"`/`"ALL"`/`"NONE"`, или 3×4 массив.
Подробнее и с примерами — в [attachments.md › Наборы рёбер](./attachments.md).

| Что нужно | Запись |
|----------|--------|
| Все рёбра | `"ALL"` (по умолчанию) |
| Только верхние | `edges=TOP` (все рёбра вокруг верхней грани) |
| Ребра вокруг граней TOP и BOT | `edges=[TOP,BOT]` |
| Все Z-выровненные (вертикальные) | `edges="Z"` |
| Конкретное ребро | `edges=TOP+FRONT` |
| Все, кроме задних | `except=BACK` |
| Комбинирование | `edges=[TOP,"Z"], except=BACK` |

## Примеры (3D-печать)

```openscad
include <BOSL2/std.scad>

// Корпус, скруглённый везде, низ — «каплей» под 45° (без поддержек)
cuboid([60, 40, 20], rounding=4, teardrop=true);

// Только верхние рёбра, угол не обрезаем → острые вертикальные кромки
cuboid([60, 40, 20], rounding=4, edges=TOP);

// Фаска 2 мм на нижних рёбрах (адгезия первого слоя), кроме правой грани
cuboid([60, 40, 20], chamfer=2, edges=BOT, except=RIGHT);

// Внешняя галтель по верху (отрицательное rounding)
cuboid([60, 40, 20], rounding=-5, edges=TOP, $fn=24);

// Куб по двум углам
cuboid(p1=[0,0,0], p2=[30,40,20]);
```

## cube() — обёртка над нативным

`cube(size, [center], [anchor], [spin], [orient])` — расширение нативного `cube()`: поддерживает анкеры и функциональную форму (`vnf = cube(...)`). `center` управляет анкером (`true`→CENTER, `false`→FRONT+LEFT+BOTTOM). **Без скруглений** — для них берите `cuboid()`.
