# Скругления и маски — детальный справочник

> Раздел: [BOSL2 — справочник](../BOSL2.md) › ключевые приёмы.
> Источник: `masks.scad`, `rounding.scad`.

Три способа сделать скругление/фаску:
1. **Встроенные аргументы** примитивов (`rounding=`, `chamfer=`, `edges=`) — см. [cuboid.md](./cuboid.md), [cyl.md](./cyl.md).
   Самый простой и быстрый — используйте первым.
2. **2D-маски + `edge_profile`** — для кастомных профилей (кромки, оги, ковы) и для рёбер,
   которые встроенные аргументы не покрывают (верх/низ призмоида, цилиндры).
3. **3D-маски + `edge_mask`/`corner_mask`** — для углов и произвольных 3D-кромок.

---

## 1. Семейство mask2d_* (2D-профили кромки)

Каждая работает и модулем, и функцией (`path = mask2d_*(...)`). Профиль ориентирован так:
угол в начале координат, одно ребро на оси X+, второе — на `mask_angle`° против часовой.
Применяется для вычитания из кромки (или как внутренний галтель).

**Общие параметры (для большинства):**

| Параметр | Назначение |
|----------|-----------|
| размер | задаётся одним из: `r`/`d` (радиус/диаметр), `h`=`height` (высота маски), `cut` (катет среза), `joint` (длина прямого участка до скругления). |
| `inset` | отступ/«борт» перпендикулярно рёбрам (скаляр или `[x,y]`). Создаёт фасонный выступ (бид). |
| `mask_angle` | угол маскируемой кромки. **В `edge_profile` автоматически = `$edge_angle`** — передавайте `mask_angle=$edge_angle`. |
| `excess` | запас маски по сторонам (чтобы покрыть кромку с нахлёстом). По умолч. 0.01. |
| `flat_top` | сделать «полку» горизонтальной (важно для тупых углов). |
| `quarter_round` | четверть окружности фиксированного радиуса — высота не зависит от `mask_angle` (сходящиеся скругления на призмоиде). |

### Типы

| Модуль | Что делает | Спец. параметры |
|--------|-----------|-----------------|
| `mask2d_roundover(r\|d\|h\|cut\|joint=, ...)` | круговое скругление/бид | `clip_angle` (клиппинг дуги от вертикали — для печатаемых низов). |
| `mask2d_smooth([mask_angle], cut=\|joint=, ...)` | скругление с непрерывной кривизной | `k` (0.5 по умолч.), `splinesteps`. Радиус не задают — только `cut`/`joint`. |
| `mask2d_teardrop(r\|d\|h\|cut\|joint=, ...)` | «капля» — ограничение свеса | `angle` (от вертикали, 45° по умолч.) — для печати без поддержек. |
| `mask2d_cove(r\|d\|h=, ...)` | ков (вогнутая четверть) | `bulge`, `quarter_round`. |
| `mask2d_chamfer(edge\|x=\|y=\|w=\|h=, [angle])` | фаска | задаётся через `edge` (длина скоса), `x` (катет по X), `y`, `w`/`width`, `h`/`height`, `angle`. |
| `mask2d_rabbet(size, ...)` | шип/выборка (ступенька). | `size`. |
| `mask2d_dovetail(edge\|width=\|height=, angle, ...)` | «ласточкин хвост» | `angle`, `shelf`. |
| `mask2d_ogee(pattern, ...)` | сложный профиль (ков+ребро) | `pattern` — строка описания. |

## 2. Применение 2D-масок к рёбрам/граням

`PARENT() module(...) CHILDREN;` — профиль экструдится вдоль рёбер родителя.

| Модуль | Назначение |
|--------|-----------|
| `edge_profile([edges], [except], [convexity]) CHILDREN;` | 2D-маска на рёбра **cuboid/призмоида/цилиндра**. Внутри переменные `$edge_angle`, `$edge_length`. |
| `edge_profile_asym(...)` | асимметричный профиль (для некорректных/направленных кромок). |
| `face_profile(faces, r\|d=, [convexity]) CHILDREN;` | профиль по всем рёбрам **вокруг граней** `faces`. |
| `corner_profile([corners], [except], [r=], [convexity]) CHILDREN;` | вращательно-экструдированная маска на углы. |

**Обязательный шаблон для призмоида** (переменные углы граней):
```openscad
diff()
  prismoid([30,20],[50,60], h=20, shift=[40,50])
    edge_profile(TOP, excess=27)
      mask2d_roundover(h=5, mask_angle=$edge_angle, $fn=128);
```
* `excess` — запас, чтобы маска перекрыла наклонные рёбра.
* `mask_angle=$edge_angle` — **обязательно**, иначе скругления не сойдутся в углах.
* Для совпадения скруглений на разных углах используйте `h=`/`height=` (одинаковая высота) или `quarter_round=true`.

## 3. 3D-маски (для углов и особых случаев)

| Модуль | Назначение |
|--------|-----------|
| `rounding_edge_mask(l, r, ...)` | скругление произвольного 3D-ребра. |
| `chamfer_edge_mask(l, chamfer, ...)` | фаска 90°-ребра. |
| `teardrop_edge_mask(...)` | скругление ребра с ограничением свеса. |
| `polygon_edge_mask(...)` | экструзия 2D-полигона в edge-маску с корректным corner-анкером. |
| `rounding_corner_mask(r)` / `chamfer_corner_mask(chamfer)` | 90°-угол. |
| `teardrop_corner_mask(...)` | угол с ограничением свеса. |
| `rounding_hole_mask(r, ...)` | скругление кромки **круглого отверстия**. |
| `rounding_cylinder_mask(r)` / `chamfer_cylinder_mask(chamfer)` | торец цилиндра. |

**Применение 3D-масок:** `edge_mask([edges]) CHILDREN;`, `face_mask([faces]) CHILDREN;`,
`corner_mask([corners]) CHILDREN;` — крепят 3D-маску к рёбрам/граням/углам родителя.

## 4. Общее скругление готовой геометрии

| Модуль | Назначение |
|--------|-----------|
| `round2d(r)` / `round2d(or=, ir=)` | скругление всех кромок 2D-фигуры (`or` — внешние, `ir` — внутренние). |
| `round3d(r)` / `round3d(or=, ir=)` | то же для 3D. Тяжёлая операция —慎重. |
| `offset3d(r, ...)` | истинный 3D-offset (расширить/сузить). |
| `shell2d(thickness, [or], [ir])` | 2D-оболочка (полый контур). |

## 5. Свободные скругления путей и призм (`rounding.scad`)

Для «органических» переходов между сечениями, недоступных встроенным аргументам:

| Модуль/функция | Назначение |
|----------------|-----------|
| `round_corners(path, [radius=]\|[cut=]\|[joint=], [method], [closed])` | скруглить углы 2D/3D пути. |
| `smooth_path(path, [tangents], [size=]\|[relsize=], [splinesteps=])` | сглаживание с непрерывной кривизной. |
| `path_join(paths, [joint], [k=])` | соединить пути со скруглениями. |
| `offset_stroke(path, [width], [rounded=], [chamfer=])` | обводка пути линией заданной ширины. |
| `offset_sweep(path, h=, [top], [bot], ...)` | выдавливание со скруглением по концам. |
| `convex_offset_extrude(height, [top], [bot])` | экструзия с offset-скруглением. |
| `rounded_prism(bottom, [top], [h=], [joint_top=],[joint_bot=],[joint_sides=], [k=])` | призма с непрерывным скруглением **вертикальных и торцевых** рёбер. ⭐ Альтернатива призмоиду со скруглением верха/низа. |

> `rounded_prism` — единственный способ непрерывно скруглить **все** рёбра призмы
> (встроенный `prismoid(rounding=)` берёт только вертикальные).
