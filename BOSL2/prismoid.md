# prismoid() / regular_prism() — детальный справочник

> Раздел: [BOSL2 — справочник](../BOSL2.md) › 3D-примитивы.
> Источник: `shapes3d.scad`. Модуль и функция (`vnf = prismoid(...)`).

Призмоид (усечённая призма с прямоугольными сечениями) и правильная N-гранная призма —
для конусных/пирамидальных деталей, стоек, держателей с уклоном.

## prismoid() — прямоугольный призмоид

### Использование
```openscad
prismoid(size1, size2, [h|l|height|length], [shift], [xang=], [yang=], ...);
// С фаской/скруглением вертикальных рёбер:
prismoid(size1, size2, h, [chamfer=]|[rounding=] | [chamfer1=],[chamfer2=],[rounding1=],[rounding2=]);
```

### Параметры

| Параметр | По умолч. | Назначение |
|----------|-----------|-----------|
| `size1` | — | `[ширина, длина]` **нижнего** сечения. |
| `size2` | — | `[ширина, длина]` **верхнего** сечения. `0` → точка (пирамида). |
| `h`/`l`/`height`/`length` | — | Высота. |
| `shift` | `[0,0]` | Сдвиг `[X,Y]` центра верха относительно центра низа. |
| `xang`/`yang` | — | Угол наклона в X/Y (вместо высоты или одного из размеров; может быть `[левый,правый]`). Чтобы избежать пересечения, размер можно дать как `[val,undef]`. |
| `rounding` | 0 | Радиус скругления **вертикальных** рёбер. |
| `rounding1`/`rounding2` | — | Скругление внизу/вверху вертикальных рёбер. |
| `chamfer` | 0 | Фаска вертикальных рёбер. |
| `chamfer1`/`chamfer2` | — | Фаска внизу/вверху. |
| `anchor` | **`BOTTOM`** | ⚠️ В отличие от `cuboid`/`cyl`, здесь дефолт `BOTTOM`. |
| `spin` | 0 | Поворот вокруг Z. |
| `orient` | `UP` | — |

### ⚠️ Порядок углов (важно!)
Когда `rounding`/`chamfer` заданы списком из 4 значений, порядок углов:
**`[X+Y+, X-Y+, X-Y-, X+Y-]`** — против часовой стрелки, начиная с «право-зад».

### Ключевые правила
1. Скругляются **только вертикальные рёбра**. Для верхних/нижних рёбер используйте `edge_profile()` (см. [rounding-masks.md](./rounding-masks.md)):
   ```openscad
   diff() prismoid([50,30],[30,20], h=15, rounding=[5,0,5,0])
     edge_profile([TOP+RIGHT, BOT+FRONT]) mask2d_roundover(h=5, mask_angle=$edge_angle);
   ```
2. Можно **смешивать** `chamfer` и `rounding` одновременно (в отличие от `cuboid`), задавая их по разным углам.
3. `mask_angle=$edge_angle` обязательно при переменных углах граней — иначе скругления не сойдутся.

### Примеры
```openscad
include <BOSL2/std.scad>
prismoid([35,50], [20,30], h=20);                 // усечённая пирамида
prismoid([40,40], [0,0], h=20);                   // полная пирамида
prismoid(100, 80, rounding=10, h=30);             // скруглённые вертикальные рёбра
prismoid(100, 80, rounding=[0,5,10,15], h=30);    // по-разному на каждом углу
prismoid([50,30],[20,20], h=20, shift=[15,5]);    // со сдвигом верха
```

## regular_prism() — правильная N-гранная призма

### Использование
```openscad
regular_prism(n, h|l=|height=|length=, r|d=|or=|od=|ir=|id=|side=, [realign=]);
// По концам:
regular_prism(n, h=, r1=|d1=|...=, r2=|d2=|...=, [shift=]);
// С фаской/скруглением торцов:
regular_prism(n, h, r, chamfer=|[chamfer1=],[chamfer2=], [chamfang=], [from_end=]);
regular_prism(n, h, r, rounding=|[rounding1=],[rounding2=], [teardrop=], [clip_angle=]);
// С текстурой боковых граней:
regular_prism(n, h, r, texture=, [tex_size=]|[tex_reps=], [tex_depth=], ...);
```

### Размер (можно задавать разными способами)
`r`/`or`/`d`/`od` — внешний; `ir`/`id` — внутренний (по впадинам); `side` — длина стороны.
`r1`/`r2`, `d1`/`d2`, `or1/or2`, `side1/side2` — раздельно по концам.

### Именованные анкеры
| Анкер | Где |
|-------|-----|
| `"edge0".."edgeN"` | центр бокового ребра; доступ через `EDGE(i)` |
| `"face0".."faceN"` | центр боковой грани; доступ через `FACE(i)` |
| `"top_edge0"`, `"bot_edge0"` | верхние/нижние рёбра; `EDGE(TOP,i)`, `EDGE(BOT,i)` |
| `"top_cornerN"`, `"bot_cornerN"` | углы |

`realign=true` — грань `face0` смотрит в X+ (иначе — ребро `edge0` на X+).

### Правила
1. **Фаска/скругление — только торцы** (как у `cyl`). Для скругления вертикальных рёбер → `rounded_prism()`.
2. Текстура **несовместима** с `rounding`/`chamfer` одновременно.
3. Скругление считается относительно **граней** призмы (не углов, как у `cyl`).

### Связанные
- `rect_tube(h, size, isize, wall=, [rounding=],[chamfer=])` — прямоугольная труба (полый бокс со стенкой).
- `wedge(size)` — 3D треугольный клин. `octahedron(size)` — октаэдр.
