# tube() — детальный справочник

> Раздел: [BOSL2 — справочник](../BOSL2.md) › 3D-примитивы.
> Источник: `shapes3d.scad`. Только модуль (`[ATTACHMENTS]`).

Цилиндрическая или коническая труба (кольцо с толщиной стенки). Базовый элемент для
втулок, фланцев, обечаек, бобышек под отверстия.

## Использование

```openscad
// По внешнему/внутреннему радиусу или диаметру
tube(h|l, or, ir, [center|anchor=], [realign=]);
tube(h|l, od=, id=);
// По толщине стенки (wall)
tube(h|l, or|od=|ir=|id=, wall=);
// Коническая труба
tube(h|l, ir1=|id1=, ir2=|id2=, or1=|od1=, or2=|od2=);
tube(h|l, or1=|od1=, or2=|od2=, wall=);
// Скругления/фаски (внешние o-, внутренние i-, по торцам 1/2)
tube(..., [rounding=]|[irounding=]|[orounding=]|[rounding1/2=]|[irounding1/2=]|[orounding1/2=],
         [chamfer=]|[ichamfer=]|[ochamfer=]|[chamfer1/2=]|[ichamfer1/2=]|[ochamfer1/2=],
         [teardrop=], [clip_angle=]);
```

## Параметры

| Параметр | По умолч. | Назначение |
|----------|-----------|-----------|
| `h`/`l`/`height`/`length` | 1 | Высота трубы. |
| `or`/`ir` | 1 / — | Внешний / внутренний радиус. |
| `od`/`id` | — | Внешний / внутренний диаметр. |
| `wall` | 1 | Горизонтальная толщина стенки (вместо второго размера). |
| `or1`/`or2` | = or | Внешний радиус снизу/сверху (конус). |
| `od1/od2`, `ir1/ir2`, `id1/id2` | — | То же диаметрами. |
| `circum` | `false` | `true` → отверстие **описывает** заданную окружность; иначе вписано. |
| `shift` | — | Сдвиг `[X,Y]` центра верха. |
| `ifn` | — | Число граней внутренней поверхности (отличное от внешней). |
| `rounding` | нет | Радиус скругления торцов (оба). |
| `irounding` | — | Скругление **внутренних** кромок торцов. |
| `orounding` | — | Скругление **внешних** кромок торцов. |
| `rounding1`/`rounding2` | — | Скругление нижнего/верхнего торца (оба контура). |
| `irounding1/2`, `orounding1/2` | — | Внутр./внешн. по торцам. |
| `chamfer`/`ichamfer`/`ochamfer` | нет | Фаски (аналогично rounding). |
| `chamfer1/2`, `ichamfer1/2`, `ochamfer1/2` | — | Фаски по торцам. |
| `teardrop` | `false` | Ограничить нижние скругления углом от торца (для печати). |
| `clip_angle` | нет | Клиппинг дуги скругления у нижних кромок. |
| `realign` | авт. | `false` → ребро на X+; `true` → грань на X+. Внутр. и внешн. всегда выровнены одинаково. |
| `rounding_fn` | — | Задать `$fn` для скруглений. |
| `anchor` | **`CENTER`** | См. [attachments.md](./attachments.md). |
| `spin` | 0 | Поворот вокруг Z. |
| `orient` | `UP` | Ось трубы. |

## Прикрепляемая часть `"inside"`

Труба имеет именованную часть **`"inside"`** — её внутренняя поверхность.
По умолчанию attach-операции работают с **внешней** поверхностью; чтобы крепить/анкерить внутри:

```openscad
tube(h=20, or=15, ir=10)
  attach_part("inside")        // переключаем контекст на внутреннюю поверхность
    position(TOP) cuboid([2,2,5]);
```

## Правила

1. Нужны **два** из трёх: `or/od`, `ir/id`, `wall`. Третий вычисляется.
2. Для конической трубы задавайте парные `_1`/`_2` параметры (например, `or1=`, `or2=`, `wall=`).
3. Префиксы `o`/`i` уточняют, к какому контуру применяется скругление/фаска; суффиксы `1`/`2` — к какому торцу.
4. `realign` устанавливается автоматически от `circum`; внешняя и внутренняя поверхности всегда синхронны.

## Примеры

```openscad
include <BOSL2/std.scad>

// Втулка 20×10, высота 15, со скруглением внешних кромок
tube(h=15, od=20, id=10, orounding=2);

// Фланец: внешние кромки скруглены, внутренние — с фаской (снизу «капля» для печати)
tube(h=8, od=30, id=6, orounding=2, ichamfer1=1, teardrop=true);

// Коническая труба по wall
tube(h=20, or1=15, or2=8, wall=2);

// Сетка отверстий через diff() + tag("remove") zcyl(... )
```

## Типичный сценарий — бобышка с отверстием

Если нужно тело **вокруг** отверстия (а не труба), обычно собирают `cyl()`/`cuboid()` и
вычитают `zcyl()` через `diff() { ...; tag("remove") zcyl(...); }`. `tube()` же уместен,
когда сама тонкостенная втулка и есть деталь.
